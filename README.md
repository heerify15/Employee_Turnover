# Employee Turnover Prediction using Logistic Regression

## 📌 Project Overview

This project focuses on predicting employee turnover using Machine Learning classification techniques. The objective is to build models that can classify whether an employee is likely to leave the organization based on various employee-related features.

The project demonstrates the complete machine learning workflow, including data preprocessing, model training, regularization, and model evaluation.

---

## 🛠️ Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib
* Scikit-learn

---

## ⚙️ Data Preprocessing

* Checked for missing values
* Identified and removed duplicate rows
* Split the dataset into training and testing sets

---

## 🤖 Models Implemented

* Baseline Logistic Regression
* Logistic Regression with L1 (Lasso) Regularization
* Logistic Regression with L2 (Ridge) Regularization

---

## 📊 Model Evaluation

The models were evaluated using the following metrics:

* Accuracy Score
* Confusion Matrix
* Classification Report
* ROC-AUC Score

---

## 🎯 Key Learnings

* Understanding binary classification using Logistic Regression
* Applying L1 (Lasso) and L2 (Ridge) regularization
* Understanding the effect of the regularization parameter (`C`)
* Data preprocessing techniques
* Evaluating classification models using multiple performance metrics
